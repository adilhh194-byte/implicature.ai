# 🧠 Implicature AI

**Extract Hidden complaints from customer reviews using linguistics and offline NLP.**

Implicature AI is a high-performance, privacy-first review analyzer designed to detect "Hidden Complaints" (concessive/adversative markers) where customers might bury a grievance behind a positive statement.

## 🚀 Key Features

- **Advanced NLP Pipeline**: A multi-stage heuristic engine that uses discourse analysis, concessive constructions, and negation-aware semantic scoring.
- **100% Offline & Privacy-First**: No data ever leaves your browser. All analysis happens via Web Workers on your local machine.
- **Rich Aesthetics**: Stunning glassmorphism UI with fully functional Light (Day) and Dark (Night) modes.
- **Multi-Format Uploads**: Support for CSV, XLSX, JSON, TXT, DOCX, and even Image uploads (via OCR).
- **Pro Performance**: Processes 2,000+ reviews in under 10ms with zero UI freezing.

## 🛠️ Tech Stack

- **Framework**: Vite + React
- **Styling**: Tailwind CSS (Glassmorphism & Day/Night Mode)
- **NLP**: Pure JavaScript Rule-Based Engine (No ML overhead)
- **File Parsing**: PapaParse (CSV), SheetJS (XLSX), Mammoth (DOCX)
- **OCR**: Tesseract.js

## 💻 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Run Diagnostics/Tests**:
